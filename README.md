# ジョブ管理ツール

プレゼンテーション用のジョブ管理ツールのフロントエンド画面です。

## 機能

このアプリケーションは以下の画面を含みます:

1. **ダッシュボード** - 全体の状況を一目で把握
2. **ジョブ一覧** - ジョブの検索・絞り込み・一括操作
3. **ジョブ詳細** - 個別ジョブの詳細情報（タブ構成）
4. **ジョブ登録・編集** - ジョブの作成と変更
5. **実行履歴** - 全ジョブの実行履歴を横断検索
6. **ログ閲覧** - 個別実行のログ詳細確認
7. **実行中ジョブ監視** - リアルタイム監視
8. **スケジュール管理** - カレンダービューでスケジュール俯瞰
9. **チーム管理** - チーム・権限管理
10. **ユーザー設定** - 個人設定

## 技術スタック

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- モックデータ（バックエンドなし）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

http://localhost:3000 でアプリケーションが起動します。

## ビルド

```bash
# プロダクションビルド
npm run build

# プロダクションサーバーの起動
npm start
```

## 注意事項

このプロジェクトはプレゼンテーション用です。実際のバックエンド機能はありません。
すべてのデータはモックデータとして `lib/mock-data.ts` に定義されています。
